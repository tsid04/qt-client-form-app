# Qt Client Form Application

This project is a C++ desktop application built using the Qt framework as part of a university course. It demonstrates the basics of GUI development, event-driven programming, and connecting user interface elements to application logic.

The application includes a graphical form interface and basic data visualization functionality.

---

## What This Project Shows

- C++ programming
- Qt framework usage
- GUI development with Qt Designer
- Signal and slot connections
- Event-driven application logic
- Basic plotting using QCustomPlot

---

## Tools Used

- C++
- Qt Framework
- Qt Creator
- QCustomPlot

---

## Notes

This project was developed as a course assignment and is intended to demonstrate foundational skills in C++ and Qt application development.
